# JavaScript Tetris
A simples Tetris game using JavaScript and the HTML5 Canvas based on [this](https://www.youtube.com/watch?v=H2aW5V46khA) tutorial.
You can play the game [here](https://mmdobal.github.io/javascript-tetris/).

# How to play
⬅ move left <br>
➡  move right <br>
⬇ move down (faster) <br>
 ⬆ rotate
